============
pyramid_skue
============

``pyramid_skue`` is a library for builidn REST API services on top of `Pyramid`_ framework.

.. _`Pyramid`: http://www.pylonsproject.org/


Installation
------------
Install with pip::

    pip install pyramid_skue

or using ``easy_install``::

    easy_install pyramid_skue
